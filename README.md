# do42-vial-rp2040
do42 vial rp2040

組裝影片
https://www.bilibili.com/video/BV1su4y1E7tG

3D列印殼件螺絲包
* 沉頭螺絲
  * M2x7  x14 上部，邊蓋，PCB
  * M2x5  x8 底部
* 銅柱
  * M2x7  x8
  * M2x9  x2 左手邊蓋
  * M2x10 x1 右手邊蓋

殼件用PLA列印的話，時間到會降解，
用其他材料列印可以放的較長久。


* 小紅點模塊鎖法
  小紅點模塊和PCB之間墊2個M2螺母，
  PCB上方直立兩個M2x9銅柱，
  兩個M2x7螺絲從小紅點模塊下方往上鎖，鎖到兩個銅柱上。
  
* 右手邊蓋處，PCB上只有一個鎖點，
  從下方用M2x7螺絲隔PCB鎖一個M2x10銅柱。
